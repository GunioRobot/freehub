6/9/2008 Release Notes

All data as of Sat June 7 is in this new system. Because this system tracks information in more detail than the old you will see some unintuitive things. For example, we don't have historic membership information so only the current membership is shown for a person. So we don't know "member since" for anyone. Moving forward we will keep historic membership information.

The primary features of this release are focused around signing people in including many enhancements to Freehub v1 including:

  • better name matching
  • see person summary when you find them
  • streamlined sign in process
  • maintain history of past memberships
  • track digging rights/EAB and classes for members
  • friendly design (thanks, Arena)
  • basic reports and user-friendly export
  • correct time zone handling
